Perlin Noise Orb

Procedural GPU Displacement

WebGL Shaders

This engine runs entirely on the GPU via custom WebGL vertex and fragment shaders. A tessellated icosahedron geometry is deformed along its vertex normals using 3D Simplex noise generated in real-time. The fragment shader calculates a high-fidelity Fresnel glow coefficient to render depth, resulting in a bioluminescent-like liquid light sphere.

Orchestration

Speed Factor 1.0x

Controls rotation increments and time scalar in procedural calculations.

Particle Density 1.0

Maps to geometry subdivision, particle count, or wave concentric channels.

Wave Displacement 1.0

Scales the amplitude of noise maps or boundary threshold distances.

// GLSL Vertex Shader Displacement
float noise = snoise(position * uNoiseFreq + vec3(0, 0, uTime * uSpeed));
vec3 displaced = position + normal * noise * uNoiseAmp;
gl_Position = projectionMatrix * modelViewMatrix * vec4(displaced, 1.0);
